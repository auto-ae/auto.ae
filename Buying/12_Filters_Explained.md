# Buying: Filters Explained

## Overview
With thousands of cars for sale, finding the perfect one requires powerful tools. **AUTO.AE** offers extensive filtering options to help you drill down by specifications, location, and condition. This guide explains how to use these filters effectively.

## Key concepts explained
| Term | Definition |
| :--- | :--- |
| **Regional Specs** | Defines which region the car was built for (GCC, American, Japanese). Essential for maintenance and cooling. |
| **Trim** | The version of the car model (e.g., Toyota Camry **LE** vs **XSE**). |
| **Warranty** | Filters cars that have an active dealer or third-party warranty. |

## Main Filters Guide

### 1. Basic Details
*   **Make & Model:** Select "Toyota", then "Land Cruiser" (or other popular models like "Nissan Patrol").
*   **Price Range:** Use the preset brackets found on the site:
    *   *Budget:* Up to 100,000 AED
    *   *Mid:* Up to 300,000 AED
    *   *Luxury:* Up to 1,000,000 AED
    *   *Exclusive:* Up to 20,000,000 AED

### 2. Technical Specs & Tags
*   **Regional Specs:**
    *   *GCC Specs:* Built for Gulf heat (larger radiators, improved AC). **Highly recommended**.

*   **Valuable Tags:** Look for listings marked with:
    *   **"Dealer maintained"** (Service history verified)
    *   **"Urgent"** (Motivated seller)
    *   **"Fully loaded"** (Top option package)

### 3. Condition & History
*   **Mileage:** Limit the max km driven.
*   **Seller Type:** Filter to see only "Dealer" stock or "Owner" listings.

## Important tips
*   **Don't Over-filter:** If you select "Red color" + "2023" + "Specific Trim", you might get 0 results. Start broad and narrow down.
*   **Use the "Verified" Filter:** Toggle this to see listings only from trusted showrooms to save time.

## Common mistakes
*   **Ignoring Import Specs:** Buying an "American Spec" car without checking its VIN. Many are salvaged/repaired cars.
*   **Confusing "Model Year" with "Registration Date":** A 2020 model might be registered in 2021. The filter uses the *Model Year*.
