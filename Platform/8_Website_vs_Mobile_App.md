# Platform: Website vs Mobile App

## Overview
AUTO.AE provides two powerful ways to access the marketplace: the responsive **Website** (accessible via any browser) and the dedicated **Mobile App** (iOS/Android). While both platforms allow you to buy, sell, and rent, they are optimized for different usage patterns. This guide helps you decide which one to use for your specific needs.

## Key concepts explained
| Term | Definition |
| :--- | :--- |
| **Responsive Web** | The website automatically adjusts its layout to fit your phone, tablet, or laptop screen. |
| **Native App** | Software installed directly on your device, allowing access to hardware features like Camera and Push Notifications. |

## Feature Comparison Table

| Feature | Website (Desktop/Mobile) | Mobile App (iOS/Android) |
| :--- | :---: | :---: |
| **Browsing Listings** | ✅ Excellent for comparing multiple tabs | ✅ Fast, infinite scroll |
| **Creating Listings** | ✅ Good for bulk uploads | ✅ **Best** (Use phone camera) |
| **Notifications** | ❌ Email only | ✅ **Instant Push Alerts** |
| **Chat** | ✅ Standard | ✅ Real-time interface |
| **Dark Mode** | ❌ Browser dependent | ✅ Native support |
| **Speed** | ⚠️ Depends on internet | ✅ Optimized caching |

## When to use which

### Use the Website when:
1.  **Comparing Prices:** You want to open 10 different listings in new tabs to compare specs side-by-side.
2.  **Dealers/Showrooms:** You are a dealer managing a large inventory csv upload.
3.  **Detailed Research:** You are reading the blog or checking detailed catalog specs.

### Use the Mobile App when:
1.  **Selling a Car:** You can walk around your car, snap photos, and post the ad in under 2 minutes.
2.  **Hunting Deals:** You want instant alerts the second a car drops in price.
3.  **Chatting:** It functions like WhatsApp, making it easier to negotiate on the fly.
